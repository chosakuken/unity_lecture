Unity 宣言チェック

問題

Inspector に以下の通りに表示されるように、正しい宣言を入力してください。

解答欄

履歴

履歴一覧

使い方

チュートリアル Inspector っぽい UI が登場するので、それに対応するような宣言を入力してください。ここで、例題を解いて見ましょう。

例題

Hoge
12
　アクセス修飾子は、このサイトの場合、すべて public であるものとします。
　このときの型について考えてみましょう。表示が「12」となっているので、整数型の int を選べば適切ですね。
　最後に変数名を考えます。「Hoge」となっている訳ですが、Unityをよく見てみると、すべての変数名が大文字から始まっています。 また、C# の変数の命名規則に則っても、ここは hoge とするのが適切です。
　最後に忘れないでほしいのが、; (セミコロン)の存在です。これを踏まえると模範解答は以下の通りになります。

模範解答

public int hoge;

これ以外にも、コンパイルエラーとならない箇所でのスペース等の記号は正答となるはずです！
是非とも変数の宣言を修得しましょう！！
